# Mining Production & Fleet Analytics Dashboard
An end-to-end data analytics projects that uses SQL Server(SSMS), Excel (Power Pivot) and Power BI to analyze mining production, fleet performance, fuel consumption and equipment downtime.

## Project overview
Mining operations generate data from multiple systems which makes it difficult to monitor production performance and quipment utilization in one place. This project combines production, fuel usage, truck loads and downtime data into a single reporting solutionto help monitor operational performance and support better decision-making.

## Tools Used
- SQL Server(SSMS) -Data cleaning, storage and analysis
-  Excel(Power Query & Power Pivot) - Data modelling and interactive reporting
-  Power BI Desktop -  Dashboard creation and visualization

## Project Structure
SQL Scripts -  Data cleaning and analysis queries
Excel Dashboard - interactive reporting with Pivot Tables and Slicers
Power BI Dashboard - Multi-page dashboard for production and fleet performance

## What I Built
### SQL Server
- Cleaned and prepared the datasets
- Created tables for production truck loads, fuel usage, downtime and equipment
- Wrote SQL queries to analyze production trends and downtime

### Excel 
- Connected Excel to SQL Server using Power Query
- Built a Power Pivot data model
- Created Pivot Tables, charts and slicers for interactive reporting

### Power BI
- Imported the SQL data model
- Built dashboards showing :
    - Production performance
    - Fleet utilization
    - Fuel consumption
    - Equipment downtime
- Created DAX measures for KPIs such as utilization and production efficiency


## Key Insights
- Identified the equipment responsible for the highest production volumes
- Highlighted the main causes of the downtime of equipment
- Compared production performance across different shifts
- Tracked the fuel usage and fleet utilization to identify improvement opportunities
